# recon

This folder contains the functions and classes for **post-processing and quantitative evaluation after the MR-MOTUS based motion models have run.** It handles:

1. Dynamic frame and full motion field reconstructions (ground truth PICS and motion-field warped images).
2. Rigid motion registration.
3. Quantitative error calculations (RRMSE for rigid motion and overall motion fields).

This yields the ground truth motion, warped images, and performance metrics.
